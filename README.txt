# Sistema De Gerenciamento De Loja de Música

Projeto para criação de um sistema com o intuito de gerenciar uma loja de música.

Nosso sistema possui as seguintes características:

    Cadastro de clientes, produtos, e funcionários
    Registro para controle do caixa da empresa
    Controle de estoque
    Venda de produtos
    Listas de:
        clientes;
        produtos;
        funcionários cadastrados.
